# Timestamp Converter

A Python script to convert Unix timestamps and readable dates.

## Description
This project helps convert timestamps into normal date/time format and vice versa.

## Features
- Timestamp to date conversion
- Date to timestamp conversion
- Useful for logs and APIs

## Technologies Used
- Python
- datetime module

## How to Run
1. Clone the repository
2. Navigate to the project folder
3. Run:

   python timestamp_tool.py

4. Choose conversion mode

## Example

Input:
1713744000

Output:
2026-04-22 00:00:00
